# 🏠 House Price Prediction ML App

An end-to-end Machine Learning project that predicts house prices using important housing features.
The project includes data analysis, model training, and an interactive web application built with Streamlit.

---

## 📌 Problem Statement

House prices depend on multiple factors such as living area, quality of construction, basement size, garage capacity, and year built.
This project aims to build a machine learning model that predicts house prices based on these key features.

---

## ⚙️ Features Used

The model predicts house price using the following inputs:

| Feature     | Description                         |
| ----------- | ----------------------------------- |
| GrLivArea   | Above-ground living area (sq ft)    |
| OverallQual | Overall material and finish quality |
| TotalBsmtSF | Basement area (sq ft)               |
| GarageCars  | Garage capacity (number of cars)    |
| YearBuilt   | Year the house was built            |

---

## 🤖 Machine Learning Workflow

1. Data exploration and visualization
2. Handling missing values
3. Feature preprocessing
4. Model training using Scikit-Learn
5. Model evaluation
6. Saving the trained model using Pickle
7. Building an interactive web app using Streamlit

---

## 🖥️ Streamlit Web App

The project includes a Streamlit application where users can:

• Enter house details
• Adjust property features
• Get an instant predicted house price

## 🚀 How to Run the Project

Clone the repository

```
git clone https://github.com/YOUR_USERNAME/house-price-prediction.git
```

Install dependencies

```
pip install -r requirements.txt
```

Run the Streamlit app

```
streamlit run app.py
```
